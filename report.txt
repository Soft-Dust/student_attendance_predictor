1. TITLE
-------
Student Attendance Predictor using Machine Learning

2. PROBLEM STATEMENT
-------------------
Student attendance is a crucial factor in academic performance. 
Educational institutions often struggle to predict which students 
are likely to miss classes, making it difficult to implement 
proactive measures. This project aims to solve this problem by 
developing a simple yet effective prediction system.

3. OBJECTIVE
-----------
The main objective of this project is to predict whether a student 
will attend the next class based on their attendance history and 
current status. This helps educational institutions to:
- Identify at-risk students
- Plan remedial actions
- Improve overall attendance rates

4. DATASET DESCRIPTION
----------------------
The dataset used in this project contains student attendance records 
with the following characteristics:
- Total Records: 30 students
- Features: 3 predictive variables
- Target: Binary classification (Will Attend / Will Not Attend)
- Source: Simulated educational data for demonstration

The dataset includes various attendance patterns and realistic 
scenarios to ensure the model learns meaningful patterns.

5. FEATURES USED
---------------
The following features are used for prediction:
- past_attendance: Student's overall attendance percentage (0-100)
- classes_missed: Number of classes missed in the last week (0-7)
- is_sick: Current health status (0=Not Sick, 1=Sick)

These features were selected based on their direct influence on 
attendance behavior and ease of collection in real-world scenarios.

6. ALGORITHM USED
----------------
Algorithm: Logistic Regression

Reason for Selection:
- Simple and interpretable
- Suitable for binary classification problems
- Fast training time
- Good performance with small datasets
- Easy to explain in academic settings

7. WORKING OF THE MODEL
-----------------------
The model follows these steps:

Step 1: Data Preparation
- Load the dataset from data.py
- Separate features (X) and target (y)
- Split data into training (70%) and testing (30%) sets

Step 2: Model Training
- Initialize Logistic Regression with random_state=42
- Train the model on training data
- The model learns the relationship between features and attendance

Step 3: Prediction
- User provides input values for the three features
- Model processes the input using trained parameters
- Returns prediction probability and binary outcome

Step 4: Output
- Displays "Will Attend" or "Will Not Attend"
- Shows confidence level (probability)
- Displays model accuracy

8. OUTPUT DESCRIPTION
--------------------
The system provides the following outputs:

Prediction Result:
- Will Attend [with checkmark emoji] / Will Not Attend [with cross emoji]
- Probability of Attending: Value between 0.00 and 1.00
- Model Accuracy: Percentage format (e.g., 85.7%)

Input Summary:
- Attendance: User-provided percentage
- Classes Missed: User-provided number
- Sick Status: Yes/No based on user input

The output is designed to be user-friendly and easily interpretable 
by non-technical users such as teachers and administrators.

9. CONCLUSION
------------
The Student Attendance Predictor successfully demonstrates the 
application of machine learning in educational settings. The project 
achieves the following:

- Provides accurate attendance predictions
- Uses simple and interpretable features
- Maintains high accuracy with minimal complexity
- Offers user-friendly interface
- Can be easily extended for additional features

The project serves as an excellent example of how machine learning 
can be applied to solve real-world educational problems while 
remaining accessible to beginners.
